# Abra: реестр учетных записей, серверов и сайтов 

Проект в самом начале. 

Установка
1. Забираем проект
2. docker-compose up -d    
3. docker-compose exec backend bash
4. ./init (Выбираем dev или prod)
5. Идем в /common/config/main-local.php и настраиваем доступ к базе данных 
6. yii migrate
